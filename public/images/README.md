# Profile Image Folder

To use your own profile image:

1. Upload your actual image file (e.g., `my-photo.jpg`) into this `public/images` folder.
2. Open the `src/data.ts` file in your code editor.
3. Find the `image:` property (around line 14).
4. Change the URL to point to your new file like this:
   `image: "./images/my-photo.jpg",`

That's it! Your image will now appear on the website.
